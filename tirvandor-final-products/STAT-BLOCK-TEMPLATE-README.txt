D&D STAT BLOCK TEMPLATE - USAGE GUIDE
=====================================

CREATED FILES:
1. Guild_Enforcer_StatBlock_Template.docx - Ready-to-use template

HOW TO USE THIS TEMPLATE:
-------------------------

QUICK START:
1. Open the .docx file in Microsoft Word
2. Replace the placeholder text in the image area with your monster portrait
3. Edit the stats, abilities, and text to match your creature
4. Save as a new file

ADDING THE IMAGE:
-----------------
1. Click in the left cell where it says "[IMAGE PLACEHOLDER]"
2. Delete the placeholder text
3. Insert → Pictures → select your monster portrait
4. Right-click image → Size and Position
5. Set width to approximately 2.3 inches
6. Make sure "Lock aspect ratio" is checked

The Guild Enforcer portrait is located at:
D:\tirvandor-products\tirvandor-project\npc-portraits\tirvandor-monster-guild-enforcer.jpg

CUSTOMIZING THE TEMPLATE:
-------------------------

HEADER (Dark Red Bar):
- Click on the creature name
- Edit text, font size remains 22pt
- The background color is RGB(149, 51, 48) - dark red

STAT BLOCK SECTIONS:
- AC/HP/Speed: Bold labels, regular text
- Ability scores: Use Courier New font to keep alignment
- Special abilities: Bold italic names, regular text
- Actions: Bold italic names, italic for "Melee Weapon Attack:", "Hit:", etc.

COLOR CODES:
- Border: RGB(88, 42, 18) or #582A12 (dark brown)
- Header bar: RGB(149, 51, 48) or #953330 (dark red)
- Section headers: RGB(149, 51, 48) (same dark red)

FORMATTING TIPS:
---------------
1. Keep consistent spacing - 3-6pt between paragraphs
2. Use horizontal lines (─ or ═) to separate major sections
3. Font sizes:
   - Creature name: 22pt
   - Section headers (ACTIONS, LORE): 11pt
   - Body text: 9-10pt
   - Ability scores: 9pt

CREATING MULTIPLE STAT BLOCKS:
-------------------------------
1. Save this template with a new name
2. Update all content for new creature
3. Replace the portrait image
4. Save again

PARCHMENT BACKGROUND (Optional):
---------------------------------
To add parchment background texture:
1. Design tab → Page Color → Fill Effects
2. Texture tab → select "Parchment" or similar
3. OR: Insert a parchment image → Behind Text

PROFESSIONAL TIPS:
------------------
- Always spell-check before finalizing
- Verify all math (HP dice, damage, bonuses)
- Check CR matches the creature's actual power level
- Include lore that gives DMs roleplay hooks
- Add tactics for combat encounters
