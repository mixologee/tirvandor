## DESIGN NOTES

### MONSTER CREATION PHILOSOPHY

All monsters follow these principles:

1. **Base + Modification**
   - Start with SRD creature
   - Reskin for Tirvandor
   - Add 1-3 unique abilities
   - Note all changes

2. **Lore Integration**
   - Tied to world/factions
   - Explained origins
   - Campaign relevance noted

3. **Tactical Variety**
   - Different combat roles
   - Unique abilities
   - Interesting encounters

4. **Balance**
   - Appropriate for CR
   - Party-tested math
   - Clear weaknesses
