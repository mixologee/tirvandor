# MONSTERS FOLDER

This folder is for custom monster stat blocks and creature documentation.

## Purpose

Store custom monsters and creatures created for The Shattered Oaths campaign and Tirvandor world.

## What Should Go Here

- Custom monster stat blocks (markdown format)
- Reskinned SRD monsters with modifications
- Regional creature encounters
- Boss monster designs
- Monster ecology notes

## Format

Each monster file should include:
- Name and type
- Base stat block (if using SRD creature)
- Modifications (what changed from base)
- Tactics and behavior
- Lore and description
- Balance notes (CR appropriate for level)
